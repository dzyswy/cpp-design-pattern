# 1. 故事
菜单有很多菜单栏，每个菜单栏下面又有很多子菜单栏和菜单项。它们组成一个树状的结构。树杈对应菜单栏，树叶对应菜单项。          
菜单栏和菜单项都有名称，也能响应增加或者减少命令。      

名词：  
- 菜单组件:  MenuCompnent
- 菜单项：MenuItem
- 菜单栏：MenuBar

```
root
├── AE
│   ├── ae_gain
│   └── exp_time
└── AWB
    ├── bgain
    ├── CCM
    │   ├── ccm0
    │   └── ccm1
    ├── ggain
    └── rgain


```
